package-vignette

Package Purpose: The purpose of this package is to provide tools,functions, and visualizations for anyone interested in exploring health and fitness. How weight, calories, sleep, activities and their difficulty relate and impact each other.

Data: The data included in this package is multiple files ranging in datasets about sleep, activity, and weight. This datasource was found from kaggle, overall already cleaned. The dataset was cleaned again to make it more user friendly for the use of this project/package.
